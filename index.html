Introduction to Git

Git is a open source version control system(VCS) that is used to track the changes made to my code

Advantages of using Git

  • Fast and efficient
  • Distributed version control
  • Enhanced collaboration
  • Compatibility
  • Data integrity

Configuring Git

git config --global user.name "Mayowa Jamaes"

git config --global user.email "mayowa.ojojames@gmail.com"

Basic git command

  • git init: used to initialise a git repository
  • git add . : used to stage changes made to my code
  • git status : used to check the status of my code
  • git commit -m "commit message" : these is used to save the staged changes with a descriptive message
  • git remote add origin : these is used to connect our local repository to our remote repository on github
  • git push origin main : these is used to push my code from local repository to remote repository on github
